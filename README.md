## 🎧 Demo
Spelar slumpmässiga lofi-toner...
🎧 Python-program som genererar Lofi / relax-musik

Ett enkelt Python-program som skapar avslappnande toner i realtid med hjälp av matematik och ljudvågor.

## 🚀 Funktioner

Genererar lugna toner (lofi vibe)

Slumpmässiga melodier

Mjuk fade in/out för avslappning

## 🛠️ Installation

pip install numpy sounddevice

▶️ Kör programmet

python main.py

Detta projekt kombinerar:
- Matematik

Perfekt för fokus, avslappning eller meditation.
